## Features

- Logging
    - messages
    - bans/kicks/disconnects
    - voice (join/leave/screen share)
- Moderation
    - auto role assign
- Room Handling
    - auto voice creation/deletion
    - ping current voice
- AI companion (friendly/angry versions)
    - different voices/roles

## How to setup

- fill `.env` file with your data from ci folder.
- pnpm install
- pnpm start
